##Conscious Coding

Job board for developers in non-profits.

Search for jobs by **Job Title** or **Job Skills**.

Jobs are sourced from the Indeed API.

View the live app [here](http://consciouscoding.herokuapp.com/).

Built using:
- Rails 4
- Flatly Bootswatch Theme
- HAML
- Omniauth-Linkedin
- HTTParty
- Simple Form
- MongoDB/Mongoid
